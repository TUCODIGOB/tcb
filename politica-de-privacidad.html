Política de privacidad

1. Finalidad y responsable del tratamiento:

En Tu Código Base tratamos los datos personales que los usuarios facilitan al comprar un producto, rellenar un formulario o contactar a través de tucodigobase.com. Puedes escribirnos para cualquier asunto relacionado con tus datos a hola@tucodigobase.com.

El responsable del tratamiento de los datos es PURA GROUP LLC, entidad titular de la marca Tu Código Base.

2. Datos que se recopilan:

Recopilamos únicamente la información necesaria para ofrecer nuestros productos y servicios digitales, que puede incluir:

  • Nombre y apellidos.
  • Correo electrónico y teléfono.
  • Datos necesarios para determinados productos.
  • Datos técnicos del dispositivo o navegación (IP, cookies, navegador, país).

Tu Código Base no solicita datos sensibles, el usuario decide libremente qué información compartir.

3. Uso de los datos:

Utilizamos los datos personales exclusivamente para:

  • Gestionar compras, facturación y entrega de productos digitales.
  • Dar soporte o respuesta a consultas.
  • Informar sobre actualizaciones o nuevos productos relacionados con Tu Código Base.
  • Cumplir con obligaciones fiscales o legales.
  • Mejorar la experiencia de navegación y seguridad del sitio.

Tu Código Base no vende ni comparte datos personales con terceros ajenos a la marca. En caso de requerirse soporte técnico o facturación a través de herramientas externas (como pasarelas de pago o plataformas de email), Tu Código Base garantiza que dichos proveedores cumplen con estándares internacionales de seguridad y confidencialidad.

4. Base legal:

El tratamiento de los datos se realiza de forma legítima porque:

  • El usuario acepta esta Política de Privacidad al enviar sus datos o completar una compra mediante acción afirmativa.
  • Existe obligación legal de conservar cierta información con fines fiscales o administrativos.

Se aplica el interés legítimo de Tu Código Base en mantener la seguridad del sitio y ofrecer un servicio de calidad.

5. Conservación de los datos:

Los datos se mantienen solo el tiempo necesario para cumplir su finalidad. Ciertos datos se conservarán durante el plazo exigido por ley para fines fiscales o administrativos. El usuario puede solicitar en cualquier momento la eliminación o modificación de sus datos.

6. Seguridad y confidencialidad:

Tu Código Base adopta medidas técnicas y organizativas razonables para proteger los datos personales frente a accesos no autorizados, pérdida o alteración. Aun así, ningún sistema es totalmente invulnerable; el usuario acepta usar el sitio de forma responsable.

7. Transferencia internacional de datos:

Tu Código Base forma parte de PURA GROUP LLC, registrada en Wyoming (Estados Unidos). El tratamiento de los datos personales se realiza conforme a la normativa europea de protección de datos (RGPD), aplicando las medidas de seguridad y confidencialidad legalmente exigidas.

8. Derechos del usuario:

En cualquier momento, el usuario puede solicitar:

  • Acceder a sus datos personales.
  • Modificarlos o actualizarlos.
  • Eliminar su información.
  • Limitar u oponerse a su tratamiento.
  • Retirar el consentimiento en cualquier momento.

Solo es necesario enviar un correo a hola@tucodigobase.com, indicando la solicitud. Las peticiones se atenderán en el menor plazo posible.

9. Enlaces externos:

Este sitio puede contener enlaces a otras páginas web, Tu Código Base no se hace responsable de las políticas de privacidad o prácticas de dichos sitios externos. Se recomienda revisar sus políticas antes de proporcionar datos personales.

10. Cambios en la política:

Tu Código Base se reserva el derecho a modificar la presente Política de Privacidad en cualquier momento. Las actualizaciones se publicarán en esta misma página y serán aplicables desde su fecha de publicación.
